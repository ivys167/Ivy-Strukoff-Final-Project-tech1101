CSS properties

Properties are things in css used to change the styling of the webpage. Below is a table of the most essential ones to remember

Property Description
align-content Specifies alignment between the lines inside a flexible container when the items do not use all avaliable space
align-items Specifies alignment for items inside a flexible container
animation Shorthand property for all the animation-* properties, animates an element of a page
background Shorthand property for all the background-* properties, inserts an image in the background of a container
border Shorthand property for border-width, border-style, and border-color properties, specifies the type of border, the size and color.
box-shadow Adds a shadow to an element, can specify color and opacity
color Sets color of text
flex Shorthand property for flex-grow, flex-shrink, and flex-basis, makes an item flexible
font-family Specifies the font familt for the text
font-size Specifies the font size of the text
justify-content Specifies alignment between the items inside a flexible container when the items do not use all available space
justify-items Is set on the grid container. Specifies the alignment of grid items in the inline direction
margin Adds space outside the element to seperate from other elements
opacity Sets opacity level for an element
padding Adds space inside the element to seperate it from other elements
text-align Specifies horizontal alignment of the text
