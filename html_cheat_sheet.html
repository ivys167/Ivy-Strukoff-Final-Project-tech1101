HTML tags

Tags are essential components in HTML that are used to format and organize text on a webpage. They give web browsers instructions about how to render links, images, text, and other content.

Below is a table of the most common and useful HTML tags to remember:

Tag Description Syntax
anchor Used to create a hyperlink on a webpage, almost always includes an href attribute <a href="url">...</a>
body Used to define the main content present inside an HTML page <body>...</body>
break Inserts a single carriage return or breaks in the content, no end tag <br>
button Used to define a clickable button, this tag is often used to submit content of a form <button type="button">
code Used to define a piece of computer code <code>...</code>
comment Used to insert comments in the HTML code <!-- ... -->
div Used to create divisions of content in the web page (text, images, headers, footers, navigation bars, etc.) <div>...</div>
form Used to create a form in html, it registers information such as logging in to a profile, submitting an email address, etc. <form>...</form>
head Used to define the head portion of the document which contains information related to the document <head>...</head>
heading Used to define headings of a page, the 6 elements are h1 through to h6. h1 is the highest level and h6 is the lowest <h1>...</h1>, <h2>...</h2>, etc.
html used to define the root of HTML and XHTML documents <html>...</html>
image Used to add an image from either a local directory or online source into the web page <img src="url" alt="description of image" width="" height="">
input Used within a form element to allow users to input data <input type="value" ... />
italic Used to italicize text, used within paragraph or heading tags <i>...</i>
label Used in forms to give a label to a form element <label>...</label>
list item Used to define a list item within an ordered list or unordered list <li>...</li>
main Used to give the main information of the document <main>...</main>
meta Used to give watchwords, potrayals, author data, and other metadata that might be utilized by the program to deliver the document accurately. <meta attribute-name="value">
paragraph Used to define a paragraph <p>...</p>
quotation Used to define a short quotation <q>...</q>
script Used to define client side script <script src="link">...</script>
span Generic inline conatiner for inline elements and content <span class="">...</span>
strong Used to bold text, used within paragraph or heading tags <strong>...</strong>
table Used to create an arrangement of data in rows and columns <table>...</table>
title Used to define the title of the HTML document <title>...</title>
